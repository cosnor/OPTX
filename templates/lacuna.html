What is Lacuna?

Lacuna compares execution times between a sparse matrix in Python and in OPTX chosen method.


Get closer to the Lacuna


Select a method of representation of a sparse matrix and know how much time the execution last for an operation.

LIL

The LIL or List of Listsformat keeps a list where each element represents a row, and each row contains a list of column indices and their corresponding nonzero values.

DOK

The DOK or Dictionary Of Keys format that keeps a dictionary where the keys are index tuples and the values are the nonzero values in those positions.

CSR

The CSR or Compressed Sparse Row format uses three arrays to store data, indices and the cumulative sum of nonzero elements per row.
